5 · 觸發與互動

互動 觸發 結果
全域快捷鍵 可自訂（預設 Ctrl+Shift+Space） idle⇄listening 切換
點擊 ICON 滑鼠左鍵（orb，已放大） 同快捷鍵切換
取消 Esc（聆聽中） 丟棄錄音，回 idle
拖曳 按住 pill 水平拖移（未鎖定時） 微調位置並記憶
鎖定拖曳 右上鎖頭鈕（左1） 切換水平拖曳啟用，狀態記憶
縮小 右上最小化鈕（左2） 隱藏視窗到系統匣，程式續跑
關閉 右上 ✕ 鈕（左3） 結束整個程式（殼 + sidecar）
系統匣 右鍵 Tray 圖示 顯示/隱藏 · STT 模式 · 設定… · 結束
自訂快捷鍵 Tray「設定…」開啟設定視窗 錄製組合 → 即時生效，無需重開
headless MCP transcribe_clip / inject_text 不經 GUI 取得文字 / 注入
